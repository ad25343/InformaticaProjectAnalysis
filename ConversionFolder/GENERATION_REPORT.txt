TRANSFORMATION TYPES USED
================================================================================

1. Source Qualifier (SQ)
   - Reads from source tables
   - Filters, projections, joins (limited)
   - Present in all 50 mappings

2. Expression (EXP)
   - Custom business logic
   - Data transformations
   - Conditional logic (IIF)

3. Lookup (LKP)
   - Dimension/reference data lookups
   - Used in medium/complex mappings
   - Examples: customer, product, exchange rates

4. Aggregator (AGG)
   - GROUP BY operations
   - SUM, COUNT, AVG, MAX functions
   - Used in 8 medium/complex mappings

5. Joiner (JNR)
   - Combine multiple sources
   - INNER JOIN, LEFT OUTER, etc.
   - Used in 10+ complex mappings

6. Router (RTR)
   - Conditional output routing
   - Multiple output paths based on conditions
   - Used in 15+ complex mappings

7. Filter (FIL)
   - Row-level filtering
   - WHERE clause logic
   - Used in 6 mappings

8. Union (UNI)
   - Combine dissimilar sources
   - Used in m_fct_counterparty_exposure
